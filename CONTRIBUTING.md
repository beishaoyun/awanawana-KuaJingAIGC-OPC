# 贡献指南

感谢你对本项目的关注！欢迎贡献代码、报告问题或提出建议。

## 如何贡献

### 1. Fork 项目

点击 GitHub 页面右上角的 "Fork" 按钮。

### 2. 克隆仓库

```bash
git clone https://github.com/YOUR_USERNAME/KuaJingAIGC-OPC.git
cd KuaJingAIGC-OPC
```

### 3. 创建分支

```bash
git checkout -b feature/amazing-feature
```

### 4. 开发

```bash
npm install
npm run dev
```

### 5. 提交更改

```bash
git add .
git commit -m "feat: 添加新功能"
```

### 6. 推送到分支

```bash
git push origin feature/amazing-feature
```

### 7. 创建 Pull Request

在 GitHub 上创建 Pull Request。

## 代码规范

- 使用 TypeScript 编写代码
- 遵循 ESLint 规则
- 使用 2 空格缩进
- 提交信息使用英文或中文，格式清晰

## 提交信息格式

```
feat: 新功能
fix: 修复 bug
docs: 文档更新
style: 代码格式调整
refactor: 重构代码
test: 测试相关
chore: 构建/工具链相关
```

## 开发环境要求

- Node.js >= 20
- npm >= 9

## 联系方式

如有问题，请通过 Issue 联系。
